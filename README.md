## build html/css/js

```
$ npm install
$ npm run compile
```

## for iOS

```
$ cordova platform add ios
$ cordova build ios
$ cordova emulate ios
```

## for Android

```
$ cordova platform add android
$ cordova build android
$ cordova emulate android
```
